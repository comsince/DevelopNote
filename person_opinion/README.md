## 概述
### 技术文档
  技术文档可以说充分说明该技术的发展状况和基本形态，它能给我们带来一种宏观的理解，这种理解是一种全局性的，是一种在遇到问题
  后能够灵活利用其中的技术解决实际工作中的问题，有时候我们的无法深入的原因更重要的在于我们没有对这项技术做一个全局的理解。
  近年来，涌现了很多对国外技术文档翻译的热潮，这其实是一种想更加了解该项技术的全局架构，做一个整体的了解，在进行深入的认知。

### 技术类型
* 开源工具类项目 需要弄清器实现原理
* 开源开发工具类项目 需要对其基本原理有了解，熟悉其基本功能
